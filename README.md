# table-flip

A Clojure compojure app designed to ... flip tables!

Inspired by lyonsinbeta's app: [Table-Flip-As-A-Service (TFAAS)](https://github.com/lyonsinbeta/table-flip).

## Usage

```
lein run -m table-flip.core
```
